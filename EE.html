Efficiency Equation

The efficiency of a device is the proportion of the supplied energy that is transferred in useful ways.

It is given as either a percentage or a decimal.

The efficiency of a device can be calculated using the equation:

efficiency = useful output energy/total energy input

energy (input or output) in joules, J

efficiency = useful power output/total power input

power (input or output) in watts, W

recall and apply

efficiency as a % or decimal

There are no symbol equations for efficiency

Note: Both equations can be used to calculate efficiency and both are recall and apply

Rearrange to find:

useful output = total input x efficiency
total input = useful output /efficiency

Worked example

A machine is 60% efficient. How much useful energy would it ouput if it was supplied with 500 J?

efficiency = useful output energy/total energy input

0.6 = useful output energy / 500

useful output energy = 0.6 x 500

useful output energy = 300
