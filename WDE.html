Work Done Equation

Work done is when a force causes an object to move through a distance.

The work done by a force can be calculated using the equation:

work done = force * displacement(distance moved in the direction of the force)

W = Fs

recall and apply

work done, W, in joules, J

force, F, in newtons, N

displacement, s, in metres, m

Rearrange to find:

force - F = W/s

displacement - s = W/F

Worked example

Calculate the distance a car is moved when the force acting on the car is 4000 N and the work done is 1 MJ.

1MJ = 1000000J

W = Fs

1000000 = 4000s

s = 1000000/4000

s = 250m
